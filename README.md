
# MUJ AI Campus Assistant

An AI-powered assistant designed for Manipal University Jaipur to support students with academic insights, wellbeing analysis, and campus data understanding.

## Problem

Students often lack a centralized AI system that can answer academic queries, analyze student patterns, and provide recommendations for improving performance and wellbeing.

## Solution

We developed a Retrieval Augmented Generation (RAG) based AI assistant that:

- Analyzes student academic datasets
- Retrieves relevant information
- Generates insights for students

## Features

- Academic performance insights
- Student stress & wellbeing analysis
- AI-powered query answering

## Tech Stack

- Python
- Streamlit
- Sentence Transformers
- FAISS vector search
- Pandas

## Run Instructions

pip install -r requirements.txt
streamlit run app.py
